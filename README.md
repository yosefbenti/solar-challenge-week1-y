## Environment Setup

### Using venv
\`\`\`bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
\`\`\`

### Using conda
\`\`\`bash
conda create -n solar_env python=3.10 -y
conda activate solar_env
pip install -r requirements.txt
\`\`\`
